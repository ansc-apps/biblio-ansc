# biblio-ansc
Aplicativo simples e prático para gerenciamento de biblioteca. Desenvolvido a partir da experiência real de quem trabalha com livros e percebeu a falta de ferramentas adequadas nos programas existentes. Este app busca facilitar a organização, o acesso e o controle de acervo, pensado para atender às necessidades de usuários de bibliotecas em geral.
